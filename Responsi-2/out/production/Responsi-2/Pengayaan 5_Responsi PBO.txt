SOAL PENGAYAAN (Refleksi Polimorfisme)
Jawablah pertanyaan berikut:
1. Apa yang dimaksud dengan polimorfisme dalam OOP? Jelaskan bagaimana metode override bekerja.
2. Mengapa array bertipe DefaultMesin dapat menyimpan objek MesinMotor dan MesinTraktor?
3. Apa perbedaan override dan overload?
4. Apa keuntungan menggunakan polimorfisme dalam sistem inventaris seperti contoh di atas?
5. Bagaimana cara kerja dynamic dispatch pada Java saat memanggil method override?

Jawaban:
1. Polimorfisme merupakan pewarisan sifat, bisa berupa upcasting ataupun downcasting. Override bekerja dengan cara "menimpa" metode milik parent class dengan child class sehingga yang keluar adalah method yang ada di child class.
2. Karena dafaultMesin merupakan parent class dari kedua class tersebut.
3. Overloading terjadi di constructor, Overriding terjadi di fungsi / prosedur
4. Mengurangi penulisan kode yang sama terus-menerus
5. Dengan mengutamakan metode dari child class
